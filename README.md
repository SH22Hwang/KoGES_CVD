- binary: 0 / 1
  
  - AS1_SEX
  
  - AS1_TIED
  
  - AS1_SLPAMSF
  
  - HAS_HYPERTENSION
  
  - HAS_DIABETES
  
  - HAS_HYPERLIPIDEMI

- cnt: 연속형 변수, float
  
  - AS1_TOTALC
  
  - HOMA_IR
  
  - WAIST_AVG
  
  - BMI
  
  - KHEI90
  
  - AMED
  
  - AMED_NONALC
  
  - DASH
  
  - PRAL
  
  - NEAP
  
  - DII
  
  - RC
  
  - AI

- cath0: 범주형 변수, 계층 없음, int
  
  - AS1_MARRYA
  
  - AS1_JOBB
  
  - AS1_DRINK

- cath1: 범주형 변수 or 연속형 변수, **계층 있음**, **int**
